## Folder Structure:

- **Data/**: Contains project datasets.
  - `Raw/`: Original, unprocessed data.
  - `Processed/`: Data cleaned and modified during analysis.
  - `Final/`: Final datasets ready for use or sharing.
  
- **Maps/**: Stores map-related files.
  - `Working/`: Draft maps and intermediate versions.
  - `Final/`: Completed maps for presentation or sharing.
  
- **Documentation/**: Contains metadata, project notes, and other supporting files.
- **Scripts/**: Contains scripts or code used for analysis and data processing.
